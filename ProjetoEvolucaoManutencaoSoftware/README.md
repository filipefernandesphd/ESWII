# Projeto Evolução de Software

O objetivo deste projeto é transmitir os conceitos e um pouco da prática da **Evolução e Manutenção de Software** por meio de uma atividade gamificada. 

## Dinâmica
- Definir grupos de 2 até 3 pessoas
- Somente uma pessoa da equipe poderá ter acesso à Internet, mas **sem acesso ao código**
- O restante da equipe terá acesso ao código, mas **sem acesso à Internet**
